# Feature 07 — Export SOAP Note

## Status: PENDING (after AI Integration)

## Overview
Generate and download a formatted SOAP note document from a completed session. The export includes patient info, SOAP sections, differential diagnoses, and recommended next steps.

## Backend Endpoints
| Method | Route | Description |
|--------|-------|-------------|
| GET | `/sessions/:id/export` | Generate and return SOAP note as PDF |
| GET | `/sessions/:id/export?format=json` | Return structured SOAP data as JSON |
| GET | `/sessions/:id/export?format=txt` | Return plain text SOAP note |

## PDF Generation
Options:
| Library | Language | Notes |
|---------|----------|-------|
| `reportlab` | Python | Full control, lightweight |
| `weasyprint` | Python | HTML → PDF, good styling |
| `puppeteer` | Node | Chrome-based, pixel-perfect |

**Recommendation**: `weasyprint` — render an HTML template to PDF, easy to style.

## Frontend Integration
- "Export SOAP" button in session header
- On click: `GET /sessions/:id/export` → download PDF
- Optional: preview modal before download
